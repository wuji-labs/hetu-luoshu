> **一陰一陽之謂道** — 一陰一陽、これを道と謂う。
> — Yijing, Xici I（《易经·系辞上》）

**あなたの AI は直線で推論する。だが現実は曲がり、釣り合い、反転する。**

AI の推論訓練の多くは線形推論に支配されている——一つの入力、一つの出力、一つの結論。明快で、判定でき、速い……しかし対極が見えず、グレーが見えず、趨勢が反転するその瞬間が見えない。

**HeTu LuoShu**（河图洛书）は《易経》の推論フレームワークを AI に組み込む——**陰陽**で考え（両極を同時に握る）、**卦象**で考え（システムを組み合わせ可能な状態へ分解する）、**変化**で考える（あらゆる静的な結論は、それがどこへ向かうかを語らねばならない）。占いではない——弁証的・体系的・動的な推論の修練である。

## 問題

```
あなた:「この戦略は良いか?」
HeTu LuoShu なしの AI:「はい。」/「いいえ。」/ 一つの数値、一つの結論。
HeTu LuoShu ありの AI:「両端を握れ。これがその陽（どこで伸びるか）、
                       そしてその陰（どこにコストが潜むか）だ。
                       システムとして見ると、これには 止（艮 / 止まる）の態が欠けている。
                       趨勢:上昇——だがこの臨界点を越えれば反転する。
                       いま最も適った一手（時中）は X。判断はあなたに残る。」
```

## AI に何を教えるか

### ☯ 四つの根本原理（四底层原则）

| 原理 | 中国語 | AI の振る舞いがどう変わるか |
|-----------|---------|--------------------------|
| 体としての陰陽 | 阴阳为体 | 判断の前に必ず**両極を露わにする**——決して一面的にしない |
| 用としての卦象 | 卦象为用 | システムを一塊にせず、組み合わせ可能な基本状態へ**分解する** |
| 機としての変易 | 变易为机 | **臨界点**と反転条件を記す。静止しているものは何一つない |
| 両を執り、中を用いる | 执两用中 | **この瞬間に適った度合い**（時中）を見出し、判断を人間へ返す |

### ䷀ 八卦（八卦）— 最小限の構造言語

三本の爻（陰 —— / 陽 ——）、八つの基本状態。複雑なシステムは、組み合わせ可能な八つの性質へと写し取られる。

| 卦 | 名 | 記号 | 性 |
|---------|----|--------|--------|
| Qian | 乾 | ☰ | 健 — 創造的、剛健、上昇 |
| Dui | 兑 | ☱ | 悦 — 悦び、開かれ、交わる |
| Li | 离 | ☲ | 丽 — 付着、明晰、光 |
| Zhen | 震 | ☳ | 动 — 奮い立たせ、衝撃、動 |
| Xun | 巽 | ☴ | 入 — 浸透、柔らか、漸進 |
| Kan | 坎 | ☵ | 陷 — 深淵、危険、流れ |
| Gen | 艮 | ☶ | 止 — 静止を保つ、止まる、限界 |
| Kun | 坤 | ☷ | 顺 — 受容、従順、支える |

> 構造化された卦のデータ（五行 / 方位 / 象 / 古典の引用）はグループの SSOT に置かれている。
> `shared/daotong/data/gua-knowledge.json`（読み取り専用の参照）。

物极必反   極まれば、必ず反する。

## 直線が、曲がる現実と出会う

HeTu LuoShu は形式論理を置き換えはしない。それを**完成させる**。

| 線形／西洋的デフォルト | + HeTu LuoShu | = 完成した推論 |
|--------------------------|---------------|------------------------|
| 一つの結論（是／否） | 陰陽：両極を握る | おのれの対極をも見据えた判断 |
| 平板な分解 | 八卦：組み合わせ可能な状態 | 再構成できる構造 |
| 静的な結論 | 変易：趨勢＋臨界点 | どこへ向かうかを知る答え |
| 平均／妥協 | 時中：その瞬間にふさわしい度合い | 状況に適った均衡 |

## 占いではない — 免責事項

このスキルが提供するのは**推論フレームワークのみ**（弁証的・体系的・動的）である。吉凶、未来の出来事、運命の予測は**一切行わず**、医療・投資・法律その他あらゆる専門的判断の根拠と**ならない**。古典の引用は概念と方法を説明するためのものである。最終的な判断——そして責任——はあなたに残る（执两用中：決めるのはあなただ）。
